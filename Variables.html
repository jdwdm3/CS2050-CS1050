Data Types

Integer, double, float, and char, are all various data types that we will be working with in C this semester. Each data type has different rules for what values it can and cannot contain. Lets go through some of the basics we need to know about these data types:

  1. int:

    int is short for integer. Integers can hold numerical values ranging from -2,147,483,648 to 2,147,483,647. The value that an integer data type can hold must be a hole number (no decimal).

    We can print out integers as follows: printf("%d",x); where %d is telling printf that x is an integer.

  2. float:

    floats can hold numerical values ranging from 1.2E-38 to 3.4E+38. The value that a float data type can be a either a whole number or a decimal. Floats are accurate to 6 decimal places

    We can print out floats with the following: printf("%f",x); where %f is telling printf that x is a float. We can tell printf how many decimals to print out as follows: printf("%.2f",x); where the .2 is telling printf to only display the first two decimals.

  3. double:

    doubles can hold numerical values ranging from 2.3E-308 to 1.7E+308. The value that a double data type can be a either a whole number or a decimal. Doubles are accurate to 15 decimal places.

    Doubles work the same as floats, except instead of %f we use %lf.

  4. char:

    char can hold single letters. Not really too much else to say about that.

    We can print out a character variable as follows: printf("%c",x); where %c is telling printf that x is a character.

Now the important thing to take from this list is knowing the following: int's hold hole numbers, floats hold hole numbers or decimals, doubles are twice(ish) the size as floats, and have the same characteristics as floats. Well and for char, it holds letters. As well as how we are going to use printf() to print the values of the variables out.

Variables

Variables are the core of programming. Variables allow you to create instances of specific data types and store them for future use. Let's go ahead and provide a quick example. Let's say I want to write a program that adds two numbers together, and these numbers are hole numbers. We know that since we are dealing with hole numbers that we won't be using char data type. We also know that floats and doubles both deal with decimal values primarily. So we are going to chose on the data type int:

Variable Declaration

int numberOne;
int numberTwo;

These two lines in C are two examples of declaring a varible. We have told our program that we are going to refer to the variables numberOne and numberTwo as integers, meaning these variables can be assigned hole numbers.

Variable Assignment

numberOne=10;
numberTwo=5;

We have now told our system that during our program, whenever we refer to numberOne we are actually refering to the number 10 (until another assignment takes place). So lets add these numbers together.

Variable Math

int numberThree;
numberThree = numberOne + numberTwo;
printf("The Answer is: %d", numberThree);
printf("Another Way to get the answer is: %d", (numberOne + numberTwo) );

Variable Initialization

Whenver we are declaring our variables, we can immediately give them a starting assignment. This starting assignment is called initializing our variable. For Example:

int anotherNumber = 0;

It is always a good practice to initialize our variables to a starting value. For integers, I always initialize the number to 0, it seems like a solid default value.
